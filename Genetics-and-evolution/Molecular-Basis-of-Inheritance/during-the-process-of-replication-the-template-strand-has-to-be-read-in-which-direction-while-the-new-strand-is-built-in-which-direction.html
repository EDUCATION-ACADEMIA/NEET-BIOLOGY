During the process of replication, the template strand has to be read in which direction, while the new strand is built in which direction?

(A) ** Built → 5’ to 3’ Read → 3’ to 5’

(B) Built → 3’ to 5’ Read → 5’ to 3’

(C) Built → 33’ to 55’ Read → 55’ to 33’

(D) Built → 55’ to 33’ Read → 33’ to 55’

EXPLANATIONS BELOW

Concept note-1: -It synthesizes the RNA strand in the 5’ to 3’ direction, while reading the template DNA strand in the 3’ to 5’ direction. The template DNA strand and RNA strand are antiparallel. RNA polymerase always builds a new RNA strand in the 5’ to 3’ direction.

Concept note-2: -Transcription and replication both occur in 5’-3’ direction. Thus, template DNA has 3’-5’ reading frame.

Concept note-3: -DNA replication always happens in the 5’→ 3’ direction. This is because the enzyme DNA polymerase, responsible for adding nucleotides to the new strands being formed, can add nucleotides only to the 3’ end of an RNA primer.

Concept note-4: -The need for accuracy probably explains why DNA replication occurs only in the 5′-to-3′ direction. If there were a DNA polymerase that added deoxyribonucleoside triphosphates in the 3′-to-5′ direction, the growing 5′-chain end, rather than the incoming mononucleotide, would carry the activating triphosphate.
