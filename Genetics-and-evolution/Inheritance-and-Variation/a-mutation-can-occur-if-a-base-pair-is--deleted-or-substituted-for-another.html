INHERITANCE AND VARIATION

[SOURCES]
A mutation can occur if a base pair is ____, deleted, or substituted for another.

(A) modified

(B) ** added

(C) transferred

EXPLANATIONS BELOW

Concept note-1: -A point mutation occurs in a genome when a single base pair is added, deleted or changed. While most point mutations are benign, they can also have various functional consequences, including changes in gene expression or alterations in encoded proteins.

Concept note-2: -Substitution. A substitution mutation occurs when one base pair is substituted for another. For example, this would occur when one nucleotide containing cytosine is accidentally substituted for one containing guanine.

Concept note-3: -A frameshift variant occurs when there is an addition or loss of nucleotides that shifts the grouping and changes the code for all downstream amino acids. The resulting protein is usually nonfunctional.

Concept note-4: -A substitution is a mutation that exchanges one base for another (i.e., a change in a single “chemical letter” such as switching an A to a G). Such a substitution could: change a codon to one that encodes a different amino acid and cause a small change in the protein produced.
