EXPLANATIONS BELOW

Concept note-1: -An insertion mutation occurs when an extra base pair is added to a sequence of bases. A deletion mutation is the opposite; it occurs when a base pair is deleted from a sequence. These two types of point mutations are grouped together because both of them can drastically affect the sequence of amino acids produced.

Concept note-2: -Types of Point Mutations. We are going to focus on the following point mutations: frameshift, silent, nonsense, and missense. Let’s start with a frameshift mutation, where there’s an insertion or deletion of one or more bases.

Concept note-3: -Frameshift Mutation A frameshift mutation in a gene refers to the insertion or deletion of nucleotide bases in numbers that are not multiples of three. This is important because a cell reads a gene’s code in groups of three bases when making a protein.

Concept note-4: -A frameshift variant occurs when there is an addition or loss of nucleotides that shifts the grouping and changes the code for all downstream amino acids. The resulting protein is usually nonfunctional.
