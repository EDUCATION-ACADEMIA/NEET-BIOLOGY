Human blood type is determined by multiple alleles (A, B, and O). Which ratio would result from a cross between a man with Type AB blood and a woman with Type O blood?

(A) 50% Type A, 50% Type O

(B) 75% Type A, 25% Type B

(C) ** 50% Type A, 50% Type B

(D) 100% Type O

EXPLANATIONS BELOW

Concept note-1: -Human blood type is determined by codominant alleles. There are three different alleles, known as IA, IB, and i. The IA and IB alleles are co-dominant, and the i allele is recessive. The possible human phenotypes for blood group are type A, type B, type AB, and type O.

Concept note-2: -The children of a man with type B blood and a woman with type AB blood could be type A, type B, and type AB, but not type O.

Concept note-3: -If the fatherâ€™s blood group is O, he has the genotype IOIO or (ii) . The mother has blood type AB, which means that her genotype would be IAIB. The possible combinations of these two groups would result in children having genotypes IAIO or IBIO, which would give them blood groups A or B respectively.

Concept note-4: -Thus, people who get a heterozygous genotype for A and B alleles have blood type AB. They make both A sugars and B sugars. There are three possible alleles for blood type, A, B, and o. However, A and B alleles are both dominant to O alleles. Thus, for a person to have type O blood, they must inherit two O alleles.
