A scientist cloned a goat. Which of these is a true statement about the cloned goat?

(A) It has new genes and traits

(B) It lacks the genes for reproduction

(C) ** It has the genes that are identical to the original goat

(D) It looks the same but has different genes.

EXPLANATIONS BELOW

Concept note-1: -Answer: C. It has the genes that are identical to the original goat.

Concept note-2: -Which of these is a true statement about the cloned goat? It has genes that are identical to the original goat. A clone is produced from the same DNA as the original organism.

Concept note-3: -Researchers have observed some adverse health effects in sheep and other mammals that have been cloned. These include an increase in birth size and a variety of defects in vital organs, such as the liver, brain and heart. Other consequences include premature aging and problems with the immune system.

Concept note-4: -A genome is an organismâ€™s complete set of genetic instructions. Each genome contains all of the information needed to build that organism and allow it to grow and develop.

Concept note-5: -The techniques employed in genetic engineering have led to the production of medically important products, including human insulin, human growth hormone, and hepatitis B vaccine, as well as to the development of genetically modified organisms such as disease-resistant plants.
