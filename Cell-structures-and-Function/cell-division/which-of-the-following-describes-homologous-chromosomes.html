Which of the following describes homologous chromosomes?

(A) ** “one from mom and one from dad”

(B) identical copies of each other

(C) ** Same “kind” of chromosome, but the exact versions of their genes might be different

(D) Chromosome 7 from one parent, chromosome 9 from the other

EXPLANATIONS BELOW

Concept note-1: -The chromosomes in a homologous pair each have the same genes in the same order, but there may be variation between them, resulting in different alleles. Some of the variants in these alleles might have functional implications while others will not.

Concept note-2: -Which of the following describes homologous chromosomes most accurately? Chromosomes that have been inherited from both the parents and which have the same genes arranged in the same order.

Concept note-3: -Answer and Explanation: Homologous chromosomes contain the same genes in the same locations, but the versions of these genes can be different. These different versions are called alleles, meaning the answer is true.

Concept note-4: -The two chromosomes in a homologous pair are very similar to one another and have the same size and shape. Most importantly, they carry the same type of genetic information: that is, they have the same genes in the same locations. However, they don’t necessarily have the same versions of genes.

Concept note-5: -Different versions of the same variant are called alleles. For example, a SNP may have two alternative bases, or alleles, C and T4.
